Discord Staff Recruitment Bot - Зависимости

Для работы бота необходимо установить следующие пакеты:

1. Node.js (версия 16.9.0 или выше)
   - Скачать с: https://nodejs.org/

2. Основные зависимости (устанавливаются командой npm install):
   - discord.js@^14.14.1 - Основная библиотека для работы с Discord API
   - dotenv@^16.3.1 - Загрузка переменных окружения

3. Дополнительные зависимости для разработки:
   - nodemon@^3.0.2 - Автоматический перезапуск при изменениях

Установка зависимостей:
1. Откройте командную строку в папке с ботом
2. Выполните команду: npm install
3. Дождитесь завершения установки

Альтернативная установка:
- Если у вас yarn: yarn install
- Если у вас pnpm: pnpm install

Проверка установки:
- node --version (должна быть 16.9.0+)
- npm --version (должна быть 8.0.0+)

После установки зависимостей:
1. Скопируйте env.example в .env
2. Заполните .env своими данными
3. Запустите бота: npm start

Поддержка: https://svortex.ru
